# 🖖 Vue Galaxy Project

Um projeto desenvolvido com **Vue.js** focado em componentização modular e estilização moderna. O foco desta aplicação foi construir uma interface de boas-vindas estruturada, utilizando as melhores práticas do ecossistema Vue para criar componentes reutilizáveis e escaláveis.

---

## 🚀 Diferenciais Técnicos

Este projeto demonstra a transição para frameworks de alta performance e a organização de uma UI componentizada:

* **Arquitetura Vue.js:** Estruturação completa utilizando o sistema de componentes do Vue, separando responsabilidades entre componentes globais e locais.
* **Componentização Modular:** Desenvolvimento de peças independentes como `Welcome`, `HelloWorld` e `Item`, facilitando a manutenção e o reuso de código.
* **Sistema de Ícones Customizado:** Criação de componentes específicos para ícones, evitando dependências pesadas de bibliotecas externas e garantindo performance.
* **Estilização Global e Escopada:** Implementação de uma base sólida de estilos globais combinada com estilos específicos para cada componente, garantindo que o layout seja consistente e livre de efeitos colaterais.

---

## 🛠️ Tecnologias Utilizadas

* **Vue.js 3:** Framework progressivo para construção de interfaces de usuário.
* **Vite:** Ferramenta de build ultra-rápida para o ambiente de desenvolvimento.
* **JavaScript (ES6+):** Lógica moderna para manipulação de dados e componentes.
* **CSS3:** Estilização avançada com foco em componentes de boas-vindas e layouts flexíveis.

---

## 📈 Histórico de Evolução

O projeto foi construído em etapas lógicas, garantindo estabilidade a cada novo recurso:

1.  **Setup Inicial:** Configuração do ambiente Vue e definição dos estilos globais.
2.  **Desenvolvimento de UI:** Criação dos componentes base (`HelloWorld` e `Welcome`).
3.  **Refinamento Visual:** Estruturação dos componentes de ícones e do componente de lista (`Item`).
4.  **Versão Funcional:** Finalização da primeira versão estável com todos os componentes integrados no `TheWelcome`.

---

## 📦 Como rodar a aplicação

1.  Clone o repositório:
    ```bash
    git clone git@github.com:juniorbonini/vue-core-concepts-playground.git
    ```
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o ambiente de desenvolvimento:
    ```bash
    npm run dev
    ```
